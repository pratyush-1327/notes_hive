Simple Notes App

A minimalistic notes app built with Flutter and Hive, designed for quick note-taking.

Features:

    Simple UI: Clean and intuitive interface.
    Quick Note Creation: Easily add new notes.
    Note Editing: Modify existing notes.
    Note Deletion: Remove unwanted notes.
    Offline Storage: Persistent storage using Hive.
